## نحوه نصب و اجرا

۱. مخزن را کلون کنید:
```bash
git clone https://github.com/Argh94/piano.git
```

۲. وارد پوشه پروژه شوید:
```bash
cd piano
```

۳. (در صورت نیاز) وابستگی‌ها را نصب کنید:
```bash
npm install
```

۴. پروژه را اجرا کنید:
```bash
npm start
```
یا فایل `index.html` را مستقیماً در مرورگر باز کنید.
